
1.环境
      python2.7
      opencv2
      sklearn
      numpy

2.脚本说明：
      run-test.py：测试脚本
      classifier.py:分类器训练，分类器调用
      distinguish.py:处理表格的主要脚本

3.文件夹说明：
      data-train/ :knn训练数据集
      final-result/ ：表格识别最终结果
      image/ ：原始表格图片
      modle/ ：分类器测试数据
      output/ ：表格单元格识别结果
      pice/  ：表格格切割得到的单元内数字图片
      template/ ：单元格再次切割得到的单个数字图片
      test/  :表格处理的中间图像测试结果
